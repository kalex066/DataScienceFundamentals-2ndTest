## Descripción del Proyecto
Análisis de un dataset de videojuegos para identificar tendencias de mercado regionales y predecir ventas globales.

## Tecnologías Utilizadas
Python (Pandas, NumPy)
Visualización: Matplotlib, SeabornMachine Learning: 
Scikit-learn (Regresión Lineal)

## Hallazgos Clave
Las ventas en Norteamérica explican el 82% del éxito global ($R^2 = 0.82$).
Existe una desconexión entre la crítica especializada y el volumen de ventas ($r = 0.10$).
Japón muestra un comportamiento de nicho especializado en Role Playing Genre.
